## 서비스 개요

**git주영**은 GitHub 계정의 커밋 수를 기반으로 코인을 지급받고, 그 코인으로 캡슐 가챠를 뽑아 카드를 수집하는 웹 서비스입니다.

- **1 커밋 = 1 코인** 기본 지급 (이벤트 적용 시 배율 변동)
- **1회 뽑기 = 10 코인** 기본 비용 (이벤트 할인 시 감소)
- 뽑은 카드는 **도감**에 자동 등록되며, 최초 획득 시 **NEW!** 배지 표시

## 사용자 기능

### 코인 충전 (GitHub 연동)

상단 네비게이션의 **🪙 코인** 버튼을 클릭하면 GitHub 코인 충전 모달이 열립니다.

1. GitHub 사용자명 입력
2. **코인 받기** 버튼 클릭
3. 커밋 수만큼 코인이 지급됩니다 (하루 1회 제한)

> **참고:** GitHub API 속도 제한으로 인해 최근 이벤트 기반 추정값을 사용할 수 있습니다.

---

### 뽑기 (캡슐 가챠)

`🎰 뽑기` 탭에서 실제 반다이 가샤폰 기기를 모델로 한 캡슐 머신을 조작합니다.

**뽑기 순서:**

| 단계 | 설명 |
|------|------|
| 1. 코인 투입 | 🪙 코인이 투입구로 떨어지는 애니메이션 |
| 2. 손잡이 회전 | TURN 노브가 돌아가며 기계가 진동 |
| 3. 캡슐 낙하 | 창 안의 볼 하나가 굴러 내려와 트레이로 떨어짐 |
| 4. 캡슐 탭 | 트레이의 캡슐을 탭하여 열기 |
| 5. 화면 플래시 | 하얀 섬광 후 카드 등장 |
| 6. 카드 공개 | 레어도별 파티클 이펙트와 함께 카드 공개 |

**레어도 종류:**

| 레어도 | 기본 확률 |
|--------|----------|
| ○ COMMON | 50% |
| ● RARE | 30% |
| ◆ EPIC | 15% |
| ★ LEGENDARY | 5% |

---

### 물리 엔진 볼

캡슐 창 안의 볼들은 실시간 물리 시뮬레이션으로 움직입니다.

- **중력**: 볼이 아래로 쌓임
- **탄성 충돌**: 볼끼리 부딪히며 튕김
- **원형 경계**: 창의 둥근 벽에서 반사
- **뽑기 이모션**: 뽑기 시 볼 하나가 굴러 내려오는 애니메이션 실행

---

### 도감

`📖 도감` 탭에서 수집한 모든 카드를 확인합니다.

- 레어도별 필터링
- 미수집 카드는 흐리게 표시
- 각 카드별 수집 횟수 및 최초 획득일 표시

## 관리자 기능

상단 우측 **⚙️** 버튼을 클릭하면 관리자 패널이 열립니다.

### 대시보드 (Dashboard)

- GitHub 사용자명으로 통계 조회
  - 총 커밋 수, 레포지토리 수, 총 스타 수
  - 주요 사용 언어 분포
  - 최근 활동 피드
- 서비스 현황: 총 아이템 수, 현재 코인, 총 뽑기 수

---

### 아이템 관리 (Items)

등록된 가챠 아이템의 **레어도별 확률**을 슬라이더로 조정합니다.

- 각 아이템의 획득 확률을 0% ~ 100% 사이에서 설정
- **확률 정규화** 버튼: 전체 확률 합이 100%가 되도록 자동 조정
- 레어도 색상 및 배지로 시각적 구분

---

### 코인 지급 (Coins)

현재 사용자에게 코인을 직접 지급합니다.

- 빠른 선택 버튼: +10, +30, +50, +100
- 직접 입력으로 원하는 수량 지급

---

### 공지 및 이벤트 (Comms)

**공지사항** 관리:

| 유형 | 아이콘 | 용도 |
|------|--------|------|
| info | ℹ️ | 일반 안내 |
| event | 🎉 | 이벤트 공지 |
| warning | ⚠️ | 주의사항 |
| update | 🔄 | 업데이트 안내 |

- 제목·내용 작성 후 등록
- 상단 고정(핀) 지원
- 목록에서 삭제 가능

**이벤트** 관리:

| 유형 | 효과 |
|------|------|
| pull_discount | 뽑기 비용 절감 (기본값 ÷ 설정값) |
| coin_multiplier | 코인 획득 배율 증가 |

- 이벤트 이름, 설명, 만료 일시, 배율값 설정
- 활성/비활성 토글
- 활성 이벤트는 사용자 화면 상단 배너에 자동 표시

## 기술 스택

| 영역 | 기술 |
|------|------|
| 프레임워크 | React 19 + TypeScript |
| 빌드 도구 | Vite |
| 스타일링 | CSS Modules |
| 물리 엔진 | 자체 구현 (requestAnimationFrame + 2D rigid body) |
| 데이터 저장 | localStorage (서버리스) |
| 외부 API | GitHub REST API (커밋·프로필·레포지토리 통계) |

## 데이터 저장 키 (localStorage)

| 키 | 내용 |
|----|------|
| `gachaItems` | 가챠 아이템 목록 및 확률 |
| `collectedItems` | 수집한 카드 목록 |
| `totalPulls` | 총 뽑기 횟수 |
| `coins` | 보유 코인 |
| `adminEvents` | 이벤트 목록 |
| `adminAnnouncements` | 공지사항 목록 |
| `adminGithubUsername` | 관리자 GitHub 사용자명 |
| `githubLastReceived` | 코인 마지막 수령일 (중복 방지) |
