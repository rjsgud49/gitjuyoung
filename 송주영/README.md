# 🎮 가챠 수집가 (Gacha Collector)

현실적인 가챠 기계를 시뮬레이션하고 아이템을 수집하는 React + TypeScript 애플리케이션입니다.

## ✨ 주요 기능

### 🎰 가챠 뽑기
- **현실적인 가챠 기계 UI**: 3D 회전 드럼 애니메이션
- **확률 기반 뽑기**: 희귀도별로 다른 확률로 아이템 드롭
- **결과 팝업**: 뽑은 아이템을 아름다운 팝업으로 표시
- **신규 획득 알림**: 처음 획득한 아이템에 특별한 배지 표시

### 📖 도감 시스템
- **아이템 수집**: 뽑은 아이템을 자동으로 도감에 저장
- **희귀도별 정렬**: 전설 → 에픽 → 레어 → 일반 순서로 표시
- **통계 정보**:
  - 수집 완성도 (진행도 바)
  - 희귀도별 보유 개수
  - 아이템별 보유 수량 및 획득 날짜

### 🖼️ 아이템 추가
- **이미지 업로드**: 자신의 이미지를 가챠 아이템으로 추가
- **아이템 커스터마이징**:
  - 아이템 이름 설정
  - 희귀도 설정 (일반 / 레어 / 에픽 / 전설)
  - 드롭 확률 조정 (슬라이더)
  - 미리보기 기능

### 💾 데이터 영속성
- **LocalStorage 저장**: 모든 데이터를 브라우저 로컬에 저장
- **자동 로드**: 페이지 재방문 시 이전 상태 복원

## 🛠️ 기술 스택

- **프론트엔드**: React 18
- **언어**: TypeScript
- **번들러**: Vite
- **스타일**: CSS Modules
- **상태 관리**: React Hooks (useState, useEffect, useMemo)
- **저장소**: LocalStorage

## 🚀 시작하기

### 설치
```bash
npm install
```

### 개발 서버 실행
```bash
npm run dev
```
브라우저에서 `http://localhost:5173/`을 열어보세요.

### 프로덕션 빌드
```bash
npm run build
```

## 🎯 사용 방법

### 1. 기본 아이템으로 뽑기
앱이 시작되면 미리 준비된 4가지 기본 아이템(⭐, 💎, 🔮, 👑)으로 가챠를 즐길 수 있습니다.

### 2. 아이템 추가하기
1. "🖼️ 아이템 추가" 탭 클릭
2. 이미지 선택 (여러 개 선택 가능)
3. 각 아이템의 정보 설정:
   - 이름
   - 희귀도
   - 드롭 확률
4. "아이템 추가" 버튼 클릭

### 3. 가챠 뽑기
1. "🎰 뽑기" 탭에서 뽑기 버튼 클릭
2. 드럼이 회전하며 3초간 애니메이션 재생
3. 결과 팝업에서 획득한 아이템 확인

### 4. 도감 확인
1. "📖 도감" 탭 클릭
2. 수집한 아이템 목록 확인
3. 완성도 및 희귀도별 통계 확인

## 🎨 디자인 특징

- **그라데이션 배경**: 보라색 그라데이션 배경으로 현대적인 느낌
- **섀도우 효과**: 3D 깊이감을 위한 다양한 박스 섀도우
- **반응형 디자인**: 모바일, 태블릿, 데스크탑 모두 지원
- **부드러운 애니메이션**: 회전, 팝업, 호버 효과

## 💾 데이터 저장 위치

모든 데이터는 브라우저의 LocalStorage에 저장됩니다:
- `gachaItems`: 추가된 가챠 아이템 목록
- `collectedItems`: 수집한 아이템 목록
- `totalPulls`: 총 뽑기 횟수

## 🎮 팁

1. **확률 조정**: 아이템 추가 시 확률 슬라이더로 레어 아이템의 확률을 낮게 설정하면 더 현실적입니다
2. **이미지 최적화**: 작은 크기의 정사각형 이미지(150x150px)를 사용하면 가장 좋습니다
3. **대량 추가**: 여러 이미지를 한 번에 업로드할 수 있습니다
4. **데이터 초기화**: LocalStorage를 비우면 모든 수집 데이터가 초기화됩니다
